# School List Generator

A comprehensive medical school admissions planning tool built with React.

## Features
- Academic profile management (MCAT, GPA, sGPA, State)
- School selection with real-time filtering
- Reach/Target/Undershoot classification system
- Clean Notion-style dashboard interface
- CSV export functionality

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` for development
4. Run `npm run build` for production

## Tech Stack
- React 18
- Vite
- Custom CSS with Proxima Nova typography
- CSV data processing with PapaParse
